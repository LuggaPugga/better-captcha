## 关于 Geetest

GeeTest 自适应验证码提供了智能化方案，可区分机器人与真人用户，防止欺诈与安全威胁。它由机器学习与 AI 驱动，在每次交互中都能提供流畅且安全的体验。

更多信息请访问 [Geetest 官方文档](https://www.geetest.com/adaptive-captcha)。
