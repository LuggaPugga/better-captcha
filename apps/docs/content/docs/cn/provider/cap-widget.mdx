## 关于 CapWidget

CapWidget 是一个基于 Cap.JS 小部件库构建的注重隐私的验证码解决方案。它为机器人保护提供了一个轻量级的客户端验证码小部件。

更多信息,请访问[官方 Cap.JS 文档](https://capjs.js.org/guide/widget.html)。

## 使用方法

CapWidget 使用 `endpoint` 属性而不是 `sitekey` 来指定验证码服务的 API 端点。

## 最佳实践

- **端点配置**: 确保您的 API 端点配置正确且可访问
- **客户端**: CapWidget 执行客户端验证 - 确保适当的服务器端验证
- **轻量级**: 对页面加载时间影响最小
- **开源**: 受益于社区贡献和透明度
- **定制**: 自定义小部件外观以匹配您应用程序的设计
