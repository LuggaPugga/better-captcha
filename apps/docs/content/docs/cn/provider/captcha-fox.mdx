## 关于 Captcha Fox

Captcha Fox 是一个现代的、注重隐私的验证码服务,提供机器人保护,具有可自定义的主题、多种挑战类型和广泛的国际化支持。

更多信息,请访问[官方 Captcha Fox 文档](https://docs.captchafox.com/)。

## 最佳实践

- **定制**: 利用广泛的主题定制选项来匹配您的品牌
- **国际化**: 使用 `lang` 选项提供本地化体验
- **隐私**: Captcha Fox 在设计时考虑了隐私 - 无不必要的跟踪
- **分析**: 使用内置分析了解机器人流量模式
- **自动主题**: 使用 `theme: "auto"` 自动匹配您网站的配色方案
