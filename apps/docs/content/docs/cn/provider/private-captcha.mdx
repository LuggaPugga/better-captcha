## 关于 Private Captcha

Private Captcha 是一个独立的、注重隐私的验证码服务,无需跟踪用户数据即可运行。它设计为用户友好、易于访问且完全符合 GDPR。

更多信息,请访问[官方 Private Captcha 网站](https://privatecaptcha.com/)。

## 重要说明

<Callout type="warn">
Private Captcha 只能在 `<form>` 元素内工作。请确保将组件包装在 form 标签中。
</Callout>

## 最佳实践

- **表单要求**: 始终将 Private Captcha 放置在 `<form>` 元素内
- **关注隐私**: 对于有严格隐私要求的应用程序来说非常理想
- **自托管**: 考虑自托管以获得最大控制权和数据主权
- **自动主题**: 使用 `theme: "auto"` 匹配您网站的配色方案
- **可访问性**: 在设计时考虑了可访问性,提供包容性的用户体验
