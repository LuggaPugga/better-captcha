## 关于 Friendly Captcha

Friendly Captcha 是一个注重隐私、符合 GDPR 的验证码替代方案,无需用户交互即可保护网站免受机器人攻击。它通过在后台执行加密谜题来工作,对合法用户完全不可见。

更多信息,请访问[官方 Friendly Captcha 文档](https://developer.friendlycaptcha.com/docs/v2)。

## 最佳实践

- **不可见的用户体验**: Friendly Captcha 完全不可见 - 用户永远不会看到挑战
- **隐私合规**: 从一开始就考虑了 GDPR 和 CCPA 合规性
- **原生自动主题**: 原生支持 `theme: "auto"`,实现无缝集成
- **可访问性**: 完全可访问 - 没有可能排除用户的视觉挑战
- **性能**: 对页面性能影响最小,在后台工作
- **欧盟托管**: 数据可以托管在欧盟,以获得额外的隐私合规性
