## 关于 HCaptcha

hCaptcha 是一个注重隐私的验证码服务,在平衡安全性和用户体验的同时保护您的网站免受机器人攻击,提供交互式和被动验证模式。

更多信息,请访问[官方 hCaptcha 文档](https://docs.hcaptcha.com/)。

## 最佳实践

- **隐私合规**: hCaptcha 在设计时就考虑了隐私,默认符合 GDPR/CCPA
- **货币化**: 考虑在您的仪表板中启用货币化,从验证码解决中获益
- **难度级别**: 根据您的安全需求与用户体验调整难度
- **被动模式**: 在低风险场景中使用被动模式以改善用户体验
- **自动主题**: 使用 `theme: "auto"` 自动匹配您网站的配色方案
