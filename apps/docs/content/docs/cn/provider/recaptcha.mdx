## 关于 ReCaptcha

Google reCAPTCHA 是使用最广泛的验证码服务之一,保护着数百万个网站免受垃圾邮件和滥用的侵害。它使用先进的风险分析通过各种挑战机制来区分人类用户和机器人。

更多信息,请访问[官方 Google reCAPTCHA 文档](https://developers.google.com/recaptcha)。

## 最佳实践

- **选择正确的版本**: 使用 v2 进行交互式挑战,使用 v3 进行不可见的基于分数的保护
- **优雅地处理错误**: 始终实现 `onError` 回调以处理网络故障
- **服务器端验证**: 始终在后端验证响应令牌
- **考虑用户体验**: 对于 v2 不可见版本,确保用户理解为什么他们会看到挑战
- **自动主题**: 使用 `theme: "auto"` 自动匹配您网站的配色方案
