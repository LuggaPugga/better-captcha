## About CapWidget

CapWidget is a privacy-focused CAPTCHA solution using the Cap.JS widget library. It provides a client-side CAPTCHA widget that can be integrated into your application for bot protection.

For more information, visit the [official Cap.JS documentation](https://capjs.js.org/guide/widget.html).

## Usage

CapWidget uses an `endpoint` prop instead of `sitekey` to specify the API endpoint for the CAPTCHA service.

<Tabs items={['React', 'Qwik', 'SolidJS', 'Vue', 'Svelte', 'Lit']} groupId="framework">
