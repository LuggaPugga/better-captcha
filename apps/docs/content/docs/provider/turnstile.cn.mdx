## 关于 Turnstile

Cloudflare Turnstile 是一个隐私优先的验证码替代方案,在大多数情况下无需交互式挑战即可验证用户。它利用 Cloudflare 的全球网络和威胁情报提供无缝的机器人保护。

更多信息,请访问[官方 Cloudflare Turnstile 文档](https://developers.cloudflare.com/turnstile/)。

## 最佳实践

- **默认不可见**: Turnstile 在大多数情况下设计为不可见 - 无需用户交互
- **原生自动主题**: Turnstile 原生支持 `theme: "auto"` - 无需转换
- **轻量级**: 对页面加载时间影响最小
- **隐私优先**: 无跟踪或 Cookie,非常适合注重隐私的应用程序
- **Cloudflare 集成**: 与其他 Cloudflare 服务无缝配合
