## 关于 Prosopo

Prosopo 是一个去中心化的、注重隐私的验证码服务,利用 Web3 技术通过基于区块链的验证提供机器人保护。

更多信息,请访问[官方 Prosopo 文档](https://docs.prosopo.io/)。

## 最佳实践

- **Web3 集成**: 对 dApp 和 Web3 应用程序非常理想
- **隐私优先**: 利用区块链进行保护隐私的验证
- **去中心化**: 没有中央机构控制验证码服务
- **挑战类型**: 根据您的需求在无摩擦和工作量证明之间选择
- **自动主题**: 使用 `theme: "auto"` 匹配您网站的配色方案
