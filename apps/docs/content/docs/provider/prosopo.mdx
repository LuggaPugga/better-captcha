## About Prosopo

Prosopo is a privacy-focused CAPTCHA service that provides comprehensive bot protection with Web3 integration, customizable themes, and multiple challenge types including frictionless verification and proof-of-work challenges.

For more information, visit the [official Prosopo documentation](https://docs.prosopo.io/).
